# Autohand CLI Examples

This directory contains examples and tutorials to help you get the most out of Autohand CLI.

## Tutorials

- **[Building a To-Do List App](./todo-list-tutorial.md)**: A step-by-step guide to building a React application using a custom `frontend-dev` sub-agent.

## Sample Agents

- **[frontend-dev.md](./agents/frontend-dev.md)**: A sample sub-agent definition for frontend development tasks. You can copy this to your `~/.autohand-cli/agents/` directory to use it.
